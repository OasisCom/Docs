# Error pendiente envío  

La inconsistencia es un error que dice Pendiente de envío por la DIAN en el EBFFAC.  

![](EBFFAC.png)  

La solución consiste en ejecutar la acción **910 BINT** para hacer el reenvío de manera manual.
